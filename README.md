# File-Sync-Service

Сервис для автоматической синхронизации локальных директорий с облачными хранилищами (на примере Яндекс.Диска). Построен с соблюдением принципов **SOLID**, что позволяет разработчику легко добавлять новые облачные провайдеры.

## 🚀 Ключевые особенности

* **Архитектура на интерфейсах**: Легкое расширение системы для работы с Google Drive, Dropbox и др.
* **Синхронизация в реальном времени**: Отслеживание изменений, загрузка новых версий и удаление лишних файлов из облака.
* **Безопасность**: Все конфиденциальные данные (токены) вынесены в переменные окружения.
* **Логирование**: Подробный мониторинг состояния через `loguru`.

## 🛠 Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/stackltd/File-Sync-Service.git
cd File-Sync-Service

```


2. **Установите зависимости:**

```bash
   pip install -r requirements.txt

```

3. **Создайте файл `.env**` в корне проекта для хранения секретов:

```env
   token_1=ваш_oauth_токен
   # Здесь можно добавить токены для других облаков

```

## ⚙️ Конфигурация

Управление поведением системы разделено на два уровня:

### 1. Переменные окружения (`.env`)

Используются только для хранения токенов.

### 2. Настройки провайдеров (`settings.py`)

В этом файле описывается логика подключения и пути. Пример структуры:

```python
clouds = {
    "yandex": {
        "storage": YandexDiskProvider,
        "token": os.getenv("YANDEX_TOKEN"),
        "base_url": "https://cloud-api.yandex.net/v1/disk/resources",
        "path_source": "/путь/к/локальной/папке",
        "path_dist": "имя_папки_в_облаке",
        "timeout": 5,  # частота проверки локальной папки (сек)
        "cloud_scan_time_delta": 300,  # частота сканирования облака
    }
}

```

## 📋 Использование

Запустите основной скрипт синхронизации:

```bash
python main.py yandex

```

* Чтобы **остановить** синхронизацию, создайте в локальной папке пустой файл с названием `0`. Программа перейдет в режим ожидания и удалит этот файл.
* Чтобы **возобновить** синхронизацию, создайте в локальной папке пустой файл с названием `1`. Программа возобновит слежение и удалит этот файл.
* Чтобы **остановить** приложение, создайте в локальной папке пустой файл с названием `---`. Программа корректно завершит работу и удалит этот файл.
